Pixel Paint Studio
純 HTML + JavaScript，支援離線製圖與線上數字填色，適合 GitHub Pages 與 iPad 操作。
1. 離線功能：匯入圖片、像素化、產生 JSON
流程：匯入圖片 → 命名 → 降採樣像素化 → 限制成 5~12 色 → 自動編號 → 匯出 JSON / 像素圖 / 線條底圖。
建議先用 24~40 格寬測試。格數越大，圖片越細緻；格數越小，越像傳統像素塗色書。
