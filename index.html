Pixel Paint Studio
純 HTML + JavaScript，支援離線製圖與線上數字填色，適合 GitHub Pages 與 iPad 操作。
1. 離線功能：匯入圖片、像素化、產生 JSON
流程：匯入圖片 → 命名 → 降採樣像素化 → 限制成 5~12 色 → 自動編號 → 去背保留主體 → 匯出 JSON / 像素圖 / 線條底圖。
建議先用 24~40 格寬測試。若原圖背景單純，開啟去背後可只保留主體與內部結構線；像魔術方塊這種圖，外框與每一小格線條都會保留。若去背過多或過少，可調整「去背容差」。
